## 6. Bug fix #2 — missing `format: pt` metadata

Compared metadata between original and pruned shards:

```python
from safetensors import safe_open

# Original:
with safe_open('.../OLMoE-1B-7B/model-00001-of-00003.safetensors', ...) as f:
    print(f.metadata())
# {'format': 'pt'}

# Pruned (broken):
with safe_open('.../OLMoE-1B-7B-pruned-32/model-00001-of-00003.safetensors', ...) as f:
    print(f.metadata())
# {'moe_pruner': 'true', 'source_shard': '...'}
```

Transformers requires `"format": "pt"` in the metadata header. The pruner was
writing its own keys but omitting it.

**Fix** — `moe_pruner/pruner.py`:

```python
# Before
metadata = {
    "moe_pruner": "true",
    "source_shard": str(src.root / shard_name),
}

# After
metadata = {
    "format": "pt",          # required by transformers loader
    "moe_pruner": "true",
    "source_shard": str(src.root / shard_name),
}
```

## 8. Analysis

| Prompt | Original | Pruned-62 | Verdict |
|---|---|---|---|
| Capital of France | Correct + bonus facts | **Identical** | No change |
| fibonacci | Nearly correct | Different but valid structure | Minimal change |
| MoE explanation | Correct description | Different wording, still correct | Minor drift |
| Largest planet | Jupiter, correct detail | Jupiter, slightly repetitive | Minor drift |
| Make tea | Normal then drifts | **Loops** on "The water is boiling" | Degraded |

**Key finding:** Dropping only 2 experts per layer (3.1% reduction) with
router-only random-probe stats is already enough to cause measurable
degradation on some prompts. The "tea" response collapses into a loop;
others show minor wording drift.

**Why this happens with random-probe stats:** The planner drops the 2 experts
that score lowest against *random hidden states*, not against real prompts.
Those experts may be heavily used on real user input patterns. The daemon
exists precisely to fix this — collect real routing stats first, then the
2 dropped experts will genuinely be the least-used ones on your workload,
making the quality impact as small as theoretically possible.
